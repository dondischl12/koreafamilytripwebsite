booking note

Do you need to book?

Try to book/check CatchTable first. If you cannot reserve, go early and expect a queue. For a family trip, do not make this the only dinner plan unless you are okay waiting.

sharing strategy

Should you host this on GitHub Pages?

Yes. For the trip, GitHub Pages is better than sending a downloaded file because the map, images, and links load like a real website on everyone’s phone. Create a repo, name the file index.html, push it, then enable Pages from the main branch.

Fast setup

1. Create repo: seoul-family-trip
2. Upload this file as index.html
3. Settings → Pages → Deploy from branch → main
4. Send the Pages URL to the family group chat.

Why images should work now

The first version used fragile image links. This version uses stable Wikimedia Commons image endpoints and a live OpenStreetMap/Leaflet map. If one image still fails, the card stays readable instead of collapsing.
